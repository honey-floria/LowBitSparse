一个作用于小 Transformer(如 Qwen2.5-0.5B/1.5B)的**通用压缩工具箱**:低 bit 权重量化(仿 GPTQ/AWQ 思路)+ 稀疏注意力(长文加速)+ **量化感知蒸馏**恢复精度。这是连接压缩算法和蒸馏/稀疏的桥。

- 里程碑:M1 权重量化(INT8→INT4)+精度评测 → M2 稀疏注意力(长序列加速) → M3 量化感知蒸馏恢复精度(已实测完成) → M4 消融+报告(0.5B 已收口,1.5B 关键结论已复现)
- 产出:压缩比、加速比、精度恢复曲线;统一汇总见 `results/summary.json`,最终报告见 `results/report.md`。
